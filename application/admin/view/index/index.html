使用事项:

  1. 请求接口返回url地址说明：
    2. a：判断domain表中有没有对应的域名：
    1.有：继续查找sku表中是否有对应的sku
    (1)有：返回对应的url(域名拼接url)
    (2)没有：返回跳转站对应的orders字段值
    2.没有：返回状态码3
    b：没有：返回状态码3
  2. 状态码
    3. 2：蜘蛛 =>不做处理，正常访问
    3：找不到sku =>跳转到404页面
    4：用domain表中id没有找到sku_redirect表中对应的domain_id
  3. domain表中extension字段值不带www
  4. sku_redirect表中domain_id字段关联domain表中id字段
